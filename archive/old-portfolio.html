    • Y86 processor Team Project

      Link to lab report

      Designed and verified a simple scaled down version of the processor - instruction fetch, decoder, control, ALU, registers, data memory read-write and data path components. The final product was able to execute any program written using the Y86-64 ISA. This project allowed me to learn more about how programs that are executed in a computer, and the issues around program loading and boot-loading. In addition to working on the processor components, we created timing diagrams that simulates the timing and output of each stage.
